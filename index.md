# Learning Objectives

- Develop systematic debugging and problem solving skills
- Learn about and construct useful circuits in common categories
    + Oscillators
    + Transistor (BJT, MOSFET) load drivers
        * Inductive (motors, solenoids)
        * Semiconductor (LEDs)
        * Resistive (speakers)
    + Other useful transistor subcircuits
    + Op-amp circuits
    + Linear power supplies
    + MOSFET logic circuits
- Understand passive and active device behavior, nonidealities, uses, and design considerations
    + Active
        * Op amps
        * Transistors
            - BJTs
            - MOSFETs
    + Passive
        * Resistors
        * Capacitors
        * Inductors
        * Diodes
            - Silicon/Schottky
            - Zener
